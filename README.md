# 🩺 Diabetes Prediction using Machine Learning

## 📌 Overview
This project predicts whether a person has diabetes using machine learning. It uses medical data such as glucose level, BMI, age, and other health indicators.

This is an end-to-end machine learning project including data preprocessing, model training, and evaluation.

---

## 🚀 Features
- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Machine learning model training
- Model evaluation and accuracy measurement
- Prediction using trained model

---

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📊 Dataset
The dataset contains medical diagnostic measurements used to predict diabetes.

Features include:
- Glucose level
- Blood pressure
- BMI
- Age
- Insulin
- Pregnancies

---

## ▶️ How to Run the Project

1. Clone the repository
2. Open the project folder
3. Run the Jupyter Notebook
4. Execute all cells

---

## 📈 Output
The model predicts whether a person is diabetic or not based on input features.
